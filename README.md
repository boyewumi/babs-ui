## Requirements

- Node.js 18+

## Getting Started

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/babs-ui.git
cd babs-ui

# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Run linter
npm run lint

# Preview production build
npm run preview
```
